📖 如何获取 Cookie？

1. 用 Chrome 打开 hr.xiaohongshu.com 并确保已登录
2. 按 F12 打开开发者工具 → 切到 Network 标签
3. 随便刷新一个请求，点击任意一条 → 在 Request Headers 中找到 cookie 字段
4. 复制 common-internal-access-token-prod= 后面的 token 值（或整行）粘贴到上方
5. 点「验证」确认 Cookie 有效 ✅
